LightQ Tutorials - Basic Control



Menu
Use this button to return to the main menu.
Restart level
Use this button to restart a level.
Continue
Use this button to play the next unsolved level.
Rotate Counter-Clockwise
Rotates the piece with the yellow highlight counter-clockwise. You can achieve the same by Left-Clicking on the object.
Rotate Clockwise
Rotates the piece with the yellow highlight clockwise. You can achieve the same by Right-Clicking on the object.
Editor related buttons
Select Object
The area pointed to by the arrow shows your currently selected object. You can select different objects by clicking inside this area.
Object Properties
Use this button to apply settings to objects, like color and if the piece is movable and/or rotatable.
Save Level
Use this button to save your level to your levelpack.
Load Level
Use this button to load a level from your levelpack.
